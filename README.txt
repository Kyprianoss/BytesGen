BytesGen generates a selected amount of bytes into a text file.

  Features:
   -Select if you want to generate bytes, megabytes, gigabytes or terabytes
   -Select if you want to generate an undetectable (from spam file checking) file or if you want to generate a detectable file but faster
    
If your antivirus is marking it as a virus, you can always scan it on VirusTotal (https://www.virustotal.com/gui/)
